# 💚 Cuida - LinuxBCN

**Programari lliure per ajudar en la gestió de l'ajuda i la cura dels malalts a casa.**

Cuida és una PWA per facilitar la coordinació familiar i mèdica.

## 🌐 Producció
**[cuida.linuxbcn.cat](https://cuida.linuxbcn.cat)**

## 🚀 Ús local
Inicia un servidor local amb:

    cd app
    python3 -m http.server 8080
    # Obrir http://localhost:8080
